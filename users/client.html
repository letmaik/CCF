Python Client¶

Available as part of CCF Python infra: https://github.com/microsoft/CCF/blob/master/tests/infra/jsonrpc.py.

The Checker class in ccf.py can be used as a wrapper to wait for requests to be committed.

Warning

The Python client does not yet support signed requests.
